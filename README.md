# ThemeGen

A basic frontend app for generating color palettes and starter design-system tokens.

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open the local URL shown in terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Current Features

- Color-wheel style hero section
- Harmony mode selector (`Complementary`, `Triadic`, `Tetradic`, `Monochromatic`)
- Base hue slider
- Auto-generated swatches with hex values
